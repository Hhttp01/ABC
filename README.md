# 🚀 מערכת ניהול תפעול מתקדמת

מערכת ניהול מקצועית לעסקי שירותים עם ממשק מודרני וידידותי למשתמש.

## ✨ תכונות עיקריות

- 📊 **ניהול חוזים ולקוחות** - הוספה, עריכה, מחיקה וחיפוש מתקדם
- 🗺️ **מפת טכנאים בזמן אמת** - מעקב אחר מיקום וסטטוס טכנאים
- 📈 **דוחות ורווחיות** - גרפים אינטראקטיביים וניתוח נתונים
- 📸 **גלריית תיעוד** - ניהול תמונות לפני/אחרי
- 👥 **פורטל לקוח** - ממשק נפרד ללקוחות
- 💾 **שמירת נתונים** - אחסון מתמשך עם Storage API

## 🛠️ התקנה

### דרישות מקדימות
- Node.js (גרסה 14 ומעלה)
- npm או yarn

### התקנה מקומית

```bash
# התקנת התלויות
npm install

# הרצה במצב פיתוח
npm start

# בניית גרסת ייצור
npm build
```

## 📦 טכנולוגיות

- **React 18** - ספריית UI
- **Tailwind CSS** - עיצוב ממשק
- **Lucide React** - אייקונים
- **Recharts** - גרפים ודוחות
- **Storage API** - שמירת נתונים

## 🚀 העלאה ל-Render

1. העלה את הקוד ל-GitHub
2. התחבר ל-[Render](https://render.com)
3. צור "New Static Site"
4. חבר את ה-repo
5. הגדרות:
   - Build Command: `npm run build`
   - Publish Directory: `build`

## 📱 תכונות נוספות

### ניהול חוזים
- 3 סוגי חוזים: בסיסי, מורחב, VIP
